# Killer Boba

pov: you are the boba trying to survive the straw.

A 2D mobile game built in Unity for iOS where you play as a boba pearl fighting for survival inside a bubble tea cup. Dodge ice cubes, avoid the suction of the straw, and stay afloat as long as you can.

## Tech Stack

- **Engine:** Unity
- **Language:** C#
- **Platform:** iOS
- **Style:** 2D sprite-based with animations

## Gameplay

- Play as a boba pearl trying to survive in a bubble tea cup
- Dodge obstacles like ice cubes that knock you back
- Avoid getting sucked up by the straw
- Survive as long as possible

## Running the Project

This project requires Unity to open and build from source.

1. Clone the repository
2. Open the project in Unity (check `ProjectSettings/ProjectVersion.txt` for the Unity version used)
3. Open the `Main` scene from `Assets/Scenes/`
4. Build for iOS via **File > Build Settings > iOS**

> Note: A Mac with Xcode is required to build and deploy to iOS devices.
